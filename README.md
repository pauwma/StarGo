# StarGo
StarGO es una aplicación basada en un futuro en el cuál el turismo espacial es una realidad cotidiana y accesible para todos, en esta se combina un buscador de viajes espaciales y una red social donde poder compartir la experiencia de estos, además contiene implementaciones de inteligencia artificial como por ejemplo recomendación y explicación de viajes mediante un chat con la API de OpenAI, generación de imágenes de perfil, etc.
